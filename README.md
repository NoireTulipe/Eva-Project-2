# EVA v2

Assistant IA personnel et professionnel pour une Maison d'Édition.

- Backend : Node.js + Express + Prisma + SQLite
- Frontend : React + Vite + Tailwind CSS
- App Android : Capacitor
- LLM : Gemini Flash (orchestrateur) + Gemini/Mistral (rédacteur)

## Lancer le projet
```bash
npm install
```
